Website Information

This page of the website contains useful information about the website, including my privacy and cookies’ statements.

Opinions, Ownership and Copyright

Except where I quote directly, all opinions expressed here are my own. They may or may not be correct; they are based on my understanding of the software and or other products that I discuss. If you think I’m wrong, I’d be grateful if you would contact me via my contact page.

If you want to use any of the photographs on this site you need permission from the photograph’s owner, most of the time this is me. If you are interested in using a photograph you can contact me using my contact page in the first instance.

You must not download images from my web server, under any circumstances.

Cookies and Local Data

The public version of this website does not use cookies, so there is no cookie notice. If I use cookies in the future, they will be for specific functionality (such as a sign-on page) and will be documented on this page.

This website does not track you and I do not pass any personal information supplied by you to third parties without your permission, unless I am required to do so by law. This will never change while I control this site’s content.

I do not use local storage on your browser, if this changes, the changes will be documented on this page.

Java Script

This website is aimed at people who develop software. In this day and age, I’d be surprised if any of you do not know what Java Script is. I’m assuming you do. I’m also assuming that you know how to enable and disable it.

It uses Java Script to improve site functionality. If you have turned Java Script off, some of the functionality will not be available to you.

GDPR and Privacy

This is a personal website, so GDPR does not apply. However, I will follow the GDPR rules, I’m just not going to write about them here (I have a life).

I repeat, this website does not track you and I do not pass any personal information supplied by you to third parties without your permission, unless I am required to do so by law. This will never change while I control this site’s content.

The only personal data you give me is the information you supply via my contact page and subsequent emails. This will be stored in a secure manner.
